Simulating Water Flow Dynamics Over a Real World Region

Implemented a cellular automatan to simulate water flow dynamics over a real world region from gathered satellite data (hydrological modelling), to ultimately allow investigation of potential flooding for hypotheical scenarios given a data set of a region. This was done by implementing the mathematical model from the paper "A two scale cellular automaton for flow dynamics modeling (2CAFDYM)" (H. Kassogué et al., 2017) in python using object oriented programming.
